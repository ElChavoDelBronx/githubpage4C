<!DOCTYPE html> sirve para indicar al navegador que el documento está escrito en HTML5.

<html> es la etiqueta raíz de la página.
Atributos:
lang indica el idioma del documento.
dir dirección del texto (izquierda a derecha o al revés).

<head> contiene metadatos e información que no se ve directamente.
Dentro se usan:
<meta charset="utf-8"> define la codificación.
<meta name="viewport"> hace que la página sea responsiva.
<title> muestra el título en la pestaña.
<link> conecta hojas de estilo.
<script> inserta código JS.

<body> contiene todo el contenido visible de la página.

<img> sirve para mostrar imágenes.
Atributos:
src ruta de la imagen.
alt texto alternativo.
width y height tamaño.
loading (lazy o eager) para cómo cargar la imagen.

<ul> crea listas con viñetas, <ol> listas numeradas y <li> cada elemento de la lista.
Atributos de <ol>: start número inicial, reversed orden inverso.

<h1> a <h6> son encabezados.
<h1> es el título principal, debe haber solo uno por página.
<h2> a <h6> son subtítulos en orden jerárquico.

<p> define un párrafo de texto.

<a> crea enlaces.
Atributos:
href destino del enlace.
target abre en misma pestaña o nueva (_blank).
rel relación del enlace (noopener noreferrer).
download descarga el recurso.

Atributos globales comunes:
id identificador único.
class clases para estilos o scripts.
style estilos en línea.
title texto al pasar el cursor.
data-* atributos personalizados.
